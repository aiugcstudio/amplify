# Amplify

Marketing site for Amplify (amplifyai.work). Static HTML/CSS/JS, no build step.

- `index.html` — the page
- `styles.css` — styles (warm editorial, light default + dark toggle)
- `script.js` — theme toggle, scroll reveals, lazy Calendly embed
- `netlify.toml` — deploy config (publishes the repo root)

Embeds: Wistia (hero video), Calendly (booking), Google Fonts. All load over HTTPS, so the site must be served over HTTP(S), not opened as a local file.
